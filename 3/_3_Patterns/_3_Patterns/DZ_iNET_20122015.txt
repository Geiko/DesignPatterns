Реализовать архитектуру приложений с применением паттернов проектирования:
1.Разработать структуру, оптимально подходящую для создания дерева меню. Каждый пункт меню может быть
  конечным (не содержащим дочерних элементов) и пункты меню, у которых есть подпункты. 

2.Для того, что бы написать веб-сайт, необходимо выполнить набор действий - отрисовать дизайн, написать код
  и оттестировать. Необходимо применить верную реализацию паттерна, что бы весь процесс по созданию сайта запускался
  по запуском одного метода - "Начать разработку сайта". Завершение разработки это : поиск ошибок, и исправление ошибок
  в работе сайта
